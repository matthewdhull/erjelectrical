Real-World Limitations

Although this application offers excellent simulation paired with unparalleled interactivity, it does have limitations that separate the function of this system from its real-world counterpart. Here's what this application doesn't do.

  • AC Inverter not depicted
  • Ground Service Bus not depicted
  • Shed Bus control is not available. However, the shed buses will always be powered when the GPU Selected or if there are 3 or more generators online.
  • Individual battery control not available.
  • Battery temperature information not available.
  • Backup battery control not available.
  • Several Abnormals cannot be simulated, including:
    • Generator overloads.
    • Battery over-temperature(s).
    • Limits on charging batteries. The batteries can be drained to 0 and charged back to 24.0 volts in this application.
    • Inadvertent essential power configuration on the ground.
